Programming API Reference: EduGraf

Class public abstract Visual

is IDisposable
This represents an object (or a hierarchical group of objects) that is placed at a particular position, rotation and size in the virtual world.
Property public IEnumerable<Visual> Children { get }: returns the visuals that have been added directly to this visual.
Property public Matrix4 Transform { get; set; }: the transform-matrix applied to this visual.
Property public IEnumerable<Visual> Descendants { get }: returns this visual and recursively all that have been added to it.
