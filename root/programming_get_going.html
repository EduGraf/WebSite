EduGraf Programming: Get Going

Required Operating System

EduGraf runs on Windows and Linux, but not on Apple devices. Apple only maintains WebGL in the browser, but not OpenGL for native applications. Thus, EduGraf currently lacks a Metal backend to run on Apple devices.

Development Setup

  1. Install the .NET 8 SDK.
  2. Install VisualStudio (Windows only) or the VS Code development environment and setup a C# project. You can test the development setup by creating a new folder and open a terminal in that folder with the console commands: dotnet new console dotnet build dotnet run code .
  3. In the C# project, install the EduGraf.OpenGL.OpenTK nuget package into your project via the package manager. Alternatively you can use the console command: dotnet add package EduGraf.OpenGL.OpenTK
  4. Rebuild your project.

Run EduGraf Test Program

The following code demonstrates a complete "hello world" program producing a plain triangle that can be viewed in 3D. This code serves as function test. Also, you get an idea how it looks like.
using EduGraf; using EduGraf.Cameras; using EduGraf.Tensors; using EduGraf.Geometries; using EduGraf.Lighting; using EduGraf.OpenGL.OpenTK; using EduGraf.UI; public class TriangleRendering(Graphic graphic, OrbitCamera camera) : Rendering(graphic, new Color3(0.2f, 0, 0.2f)) { private static readonly float[] Positions = [ -0.5f,-0.5f, 0, +0.5f,-0.5f, 0, +0.0f, 0.5f, 0 ]; private static readonly float[] Normals = [ 0, 0, 1, 0, 0, 1, 0, 0, 1 ]; public override void OnLoad(Window window) { var material = new EmissiveUniformMaterial(new Color4(1, 0, 0, 1)); var shading = Graphic.CreateShading([], [ material ], camera); var geometry = Geometry.Create(Positions, Normals); var surface = Graphic.CreateSurface(shading, geometry); var triangle = Graphic.CreateVisual("", surface); Scene.Add(triangle); } } static class Program { static void Main() { var graphic = new OpenTkGraphic(); var camera = new OrbitCamera(new Point3(0, 0, 2), Space.Origin3); var rendering = new TriangleRendering(graphic, camera); using var window = new OpenTkWindow("Hello World", graphic, 400, 400, camera.Handle); window.Show(rendering); } }
